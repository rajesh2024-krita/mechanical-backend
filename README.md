# Maintenance Management System Backend

A production-ready backend for Mechanical Maintenance Management System built with Node.js, Express, MySQL, and Sequelize.

## Features

- 🔐 JWT Authentication with Role-Based Access Control (RBAC)
- 👥 User Management (SUPER_ADMIN, MANAGER, TECHNICIAN)
- 🏢 Multi-building Management
- 🛠 Equipment Management with Photo Upload
- 📋 Task Assignment and Tracking
- 📊 Dashboard and Reports
- 📝 Activity Logging
- 🔄 Offline Sync Support
- 📷 File Upload with Multer
- 📚 Swagger API Documentation
- 🛡️ Security with Helmet and Rate Limiting

## Tech Stack

- Node.js
- Express.js
- MySQL with Sequelize ORM
- JWT for authentication
- Multer for file uploads
- Swagger/OpenAPI for documentation
- Winston for logging

## Prerequisites

- Node.js (v14 or higher)
- MySQL (v5.7 or higher)
- npm or yarn
